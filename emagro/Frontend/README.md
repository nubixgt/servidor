# Frontend Template

This is a clean frontend template using Vue 3 + Vite + Tailwind CSS.

## Features
- Vue 3 (Composition API)
- Vue Router (Basic setup with Auth guard placeholder)
- Pinia (State Management installed)
- Tailwind CSS (Configured with fonts)
- Axios (Installed)

## Project Setup

### Install Dependencies
```sh
npm install
```

### Compile and Hot-Reload for Development
```sh
npm run dev
```

### Build for Production
```sh
npm run build
```
